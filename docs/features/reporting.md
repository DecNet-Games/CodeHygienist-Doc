# HTML Report Generator

Accountability is everything when handling large refactoring efforts across multiple developers. Out-source studios fixing client code need proof of optimization, and large teams need metrics tracking over time.

Code Hygienist Pro includes an extensible **HTML Report Generator**.

![Report Preview Image Placeholder]({{ site.baseurl }}/assets/images/report-dashboard-placeholder.png)

## What it captures

Before you apply your Batch Fixes across the target directories, click **Generate Project Report**. Code Hygienist measures:

- **Total Found Issues**
- **Issue Distribution by Severity (Critical vs Warnings vs Info)**
- **A Cumulative Total "Hygiene Score" Percentage**.

The generated HTML file is designed to be standalone and can be attached to emails or embedded in studio Wikis for pipeline visibility.
